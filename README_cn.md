## HOW TO USE
- 需要Xposed框架和Xposed框架管理器
- 需要科学上网
- 下载并安装应用
- 重启手机, 打开权限管理(安全中心等等), 在应用权限设置中允许**悬浮窗**权限
- 打开应用, 选个你喜欢的地方
- 打开PokemonGo, 起飞~

## FEATURES
- 提供了一个简单的方向选择器, 单击分别向四个方向移动固定的距离
- GPS开关要打开, 但是实际并不会调起系统GPS
- 使用Google地图选择地点, 世界任你遨游
- 完全合法, 没有反编译或替换他们一个字节
